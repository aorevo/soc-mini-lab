# SOC Mini Lab: анализ SSH-логов Linux

## Описание проекта

Этот проект представляет собой учебную SOC-лабораторию для анализа событий безопасности в Linux. В рамках проекта был настроен SSH-сервис, сгенерированы тестовые события подключения и выполнен анализ логов.

Основной фокус проекта — поиск подозрительных SSH-событий: попыток входа под несуществующим пользователем и соединений, закрытых на этапе предварительной аутентификации.

## Цель проекта

Цель проекта — отработать базовый процесс SOC L1:

* сбор логов;
* поиск подозрительных событий;
* анализ SSH-подключений;
* описание правил обнаружения;
* оформление отчёта об инциденте;
* подготовка рекомендаций по защите.

## Используемые инструменты

* Linux VM;
* OpenSSH Server;
* `journalctl`;
* `grep`;
* `ss`;
* терминал Linux.

## Что было сделано

1. Проверена работа SSH-сервиса.
2. Проверено наличие открытого SSH-порта.
3. Сгенерированы тестовые SSH-события.
4. Найдены события `Invalid user`, `Connection closed` и `[preauth]`.
5. Сохранён обезличенный пример логов.
6. Составлены правила обнаружения.
7. Подготовлен отчёт об инциденте.
8. Сформулированы рекомендации по защите.

## Пример команды для анализа

```bash
sudo journalctl -u ssh --since "today" | grep -Ei "invalid user|connection closed|preauth"
```

## Основные выводы

В логах были обнаружены попытки SSH-подключения под несуществующим пользователем и соединения, закрытые до прохождения аутентификации.

В реальной инфраструктуре такие события могут указывать на сканирование, подбор пользователей или попытки несанкционированного доступа.

Успешного несанкционированного входа в рамках лаборатории обнаружено не было.

## Рекомендации

* использовать SSH-ключи;
* отключить парольную аутентификацию, если она не нужна;
* запретить вход под root по SSH;
* ограничить SSH-доступ через firewall;
* регулярно проверять SSH-логи;
* отслеживать серии неуспешных попыток входа.

## Безопасность

Проект выполнен только в локальной лабораторной среде.
Реальные IP-адреса, имена пользователей и имя хоста были обезличены перед публикацией.
